1. Sube todos los archivos a un nuevo repositorio en GitHub.
2. Entra a https://vercel.com → Importa tu repositorio.
3. En Settings → Environment Variables, agrega:
   SUPABASE_URL
   SUPABASE_ANON_KEY
   OPENAI_API_KEY
4. Despliega el proyecto.
5. Abre /login.html para comenzar.
6. Las conversaciones nuevas se guardarán y se evaluarán automáticamente.
